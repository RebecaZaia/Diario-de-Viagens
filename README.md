# Diario-de-Viagens
Aplicativo que permite registrar suas experiências em diferentes lugares. Nele, é possível adicionar informações como o local visitado, cidade, data da viagem, avaliação da experiência, comentários pessoais e fotos tiradas na hora.  Objetivo: ajudar a guardar e relembrar suas experiências de viagem de forma simples, prática e visual.
