# Diario-de-Viagens
O TravelLog é um aplicativo de diário de viagens que permite ao usuário registrar suas experiências em diferentes lugares. Nele, é possível adicionar informações como o local visitado, cidade, data da viagem, avaliação da experiência, comentários pessoais e fotos tiradas na hora.
O objetivo do TravelLog é ajudar o usuário a guardar e relembrar suas experiências de viagem de forma simples, prática e visual.
